# Changelog

## tseLCA 1.0.0

- Initial submission to CRAN.
- Implemented BCH and ML three-step estimators for latent class
  analysis.
- Added support for Gaussian, Poisson, and binomial distal outcome
  families.
- Included data-generating process replicating the Bakk & Kuha (2018)
  simulation study.
- Implemented sandwich variance estimation to propagate measurement
  uncertainty.
